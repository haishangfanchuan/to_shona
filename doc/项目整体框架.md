# 00-项目整体框架.md

## 一、 项目概述表

| 属性 | 说明 |
| :--- | :--- |
| **项目名称** | 「街角的话」 (Words at the Corner) |
| **核心玩法** | 横向卷轴步行模拟 + 步伐同步 + 情感选择互动 |
| **视觉风格** | 暖色调复古像素风 (32x32 角色，32 色调色板限制) |
| **平台/环境** | 移动端浏览器 (H5) / 竖屏 (逻辑分辨率 192x320) |
| **开发引擎** | Phaser 3 (v3.80+) + TypeScript + Vite |
| **核心情感体验** | 沉默中的陪伴、回忆的泛起、鼓起勇气的道歉、等待审判的局促 |

## 二、 故事线与背景设定

**背景润色：**
一段因误会和冲动而陷入冷战的感情。两人约定在傍晚散步，这是打破僵局的最后机会。谁都没有先开口，只有脚步声在熟悉的街道上回荡。路过的每一处风景都在拉扯着共同的记忆。夜色渐深，走到地铁站的尽头，离别在即，主角终于决定打破沉默，将压在心底的歉意化作漂浮的字句。

**故事线：**
> 傍晚，两人一前一后，沉默地走在铺满余晖的街道上。
> 走过曾经一起躲雨的眼镜店屋檐，路过总是买同一种口味双色冰淇淋的零食店。
> 夜色降临，城市的霓虹灯悄然亮起，远处的地铁站入口逐渐放大。
> 走累了，在地铁站前的长阶梯上并肩坐下。
> 借着微弱的星光，终于把那句排练了无数次的话，一句一句说了出来。
> 晚风吹过，等待着对方最后的宣判。

## 五、 核心机制概要

1. **步伐同步 (Pace Sync)：** 玩家通过左右滑动控制主角，需与匀速前进的 NPC 保持特定距离（-40px 到 60px）。
2. **温度条反馈 (Temperature System)：**
   * 同步状态：温度缓慢上升，画面色调变暖。
   * 落后状态：温度下降，屏幕四周出现暗角 (Vignette) 逼迫感。
   * 超前状态：NPC 停止移动并低头，温度条冻结，暗示“不要催促”。
3. **回忆触点 (Memory Triggers)：** 特定坐标区间，两人同时处于该区域时触发额外加分和剧情碎片。
4. **渐进式对话 (Progressive Dialogue)：** 第二幕后半段，抛弃传统对话框，采用点击屏幕逐句飘出文字的方式，强化“挤出话语”的艰难感。

## 五、 通用组件逻辑 (伪代码)

### PaceSyncSystem (步伐判定系统)
```typescript
// 每帧调用
function update(playerX, npcX) {
    let distance = playerX - npcX;
    
    if (distance > CONSTANTS.DISTANCE.TOO_FAST) {
        // 玩家太快
        npc.stopWalkAnimation();
        npc.playIdleLowHead(); // 低头
        temperatureBar.isFrozen = true;
    } else if (distance < CONSTANTS.DISTANCE.TOO_FAR) {
        // 玩家太慢
        npc.resumeWalk();
        vignette.setAlpha(Math.min(1, vignette.alpha + 0.02)); // 加深暗角
        temperatureBar.decrease();
    } else {
        // 同步状态
        npc.resumeWalk();
        vignette.setAlpha(Math.max(0, vignette.alpha - 0.05)); // 减轻暗角
        temperatureBar.increase();
        
        // 缓慢收缩间距到完美同步距离
        if (distance > CONSTANTS.DISTANCE.SYNC_TARGET) {
            // 在引擎逻辑中，让玩家稍微靠近NPC
        }
    }
}
```

function update(time, delta) {
    // 1. 获取输入
    let speedX = virtualJoystick.getVelocityX();

## 五、 回忆触点设计

* **条件**：`abs(player.x - memory.triggerX) < 20` 且 `abs(npc.x - memory.triggerX) < 20`。
* **流程**：

    3. 弹出全屏半透明遮罩与回忆文案。
    4. 温度条 +5。
    5. 玩家再次点击，遮罩消失，恢复行走。

## Part A: CityNightScene (夜色城市)

* **透视放大效果**：当地铁站标志 (单独一个 Sprite 层) 进入视野时，根据玩家的 X 坐标计算比例：
    ```typescript

    }
    ```

### 对话模式流程 (lineIndex 驱动)

维护变量 `let lineIndex = 0;`

```typescript
function onPointerDown() {
    if (lineIndex >= DIALOGUES.SUBWAY_APOLOGY.length) return;
    
    let textStr = DIALOGUES.SUBWAY_APOLOGY[lineIndex];
    let startX = player.x;
    let startY = player.y - 30; // 头顶位置
    
    // 创建文字对象
    let textObj = this.add.text(startX, startY, textStr, { 
        fontSize: '10px', fill: '#FFFFFF', alpha: 0 
    });
    
    // 计算目标位置 (上方信件区域，逐行排列)
    let targetY = 40 + (lineIndex * 15);
    let targetX = 96; // 屏幕中心居中
    textObj.setOrigin(0.5, 0.5);

    // 飘移动画 (Tween)
    this.tweens.add({
        targets: textObj,
        x: targetX,
        y: targetY,
        alpha: 1,
        duration: 1500,
        ease: 'Sine.easeOut'
    });

    lineIndex++;
    
    if (lineIndex === DIALOGUES.SUBWAY_APOLOGY.length) {
        triggerFinalPresentation();
    }
}
```

2. 整个文字区块组合应用轻微的呼吸动画 (Scale 0.98 <-> 1.02，时长 2000ms 循环)。
3. 根据当前温度值判定：
