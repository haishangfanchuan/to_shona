## 三、 初始化步骤清单

1. **注册系统**：实例化 `PaceSyncSystem` 和 `TemperatureBar`。
2. **加载地图**：添加 `bg_street` 图像，设置 `setOrigin(0, 0)`。
3. **添加角色**：创建 Player 和 NPC 精灵，播放各自的 Walk 动画。
4. **绑定摄像机**：`this.cameras.main.startFollow(player, true, 0.1, 0)`，限定 Bounds。
5. **UI 覆盖**：添加顶层暗角图片 `vignette` (默认 alpha 0)，添加顶层温度条。
6. **添加气泡**：Player 头顶绑定 `...` 精灵序列帧播放。

## 四、 核心更新逻辑 (Update 伪代码)

```typescript
function update(time, delta) {
    // 1. 获取输入
    let speedX = virtualJoystick.getVelocityX();
    player.setVelocityX(speedX);

    // 2. NPC 自动向前
    if (!npc.isPaused) {
        npc.setVelocityX(CONSTANTS.SPEED.NPC_WALK);
    }

    // 3. 步伐同步判定
    PaceSyncSystem.update(player.x, npc.x);

    // 4. 环境色调随进度渐变 (Day to Dusk)
    let progress = player.x / CONSTANTS.SCENE_LENGTH.STREET;
    let currentColor = Phaser.Display.Color.Interpolate.ColorWithColor(
        colorOrange, colorPurple, 100, progress * 100
    );
    bg.setTint(Phaser.Display.Color.GetColor(currentColor.r, currentColor.g, currentColor.b));

    // 5. 触点检测
    checkMemoryTriggers(player.x, npc.x);

    // 6. 场景结束判定
    if (player.x >= CONSTANTS.SCENE_LENGTH.STREET) {
        triggerSceneTransition();
    }
}
```

## 五、 回忆触点设计

### 触发条件与流程
* **条件**：`abs(player.x - memory.triggerX) < 20` 且 `abs(npc.x - memory.triggerX) < 20`。
* **流程**：
    1. 满足条件时，两人头顶同时弹出 `!` 图标。
    2. 玩家此时点击屏幕，游戏暂停 (Physics 冻结)。
    3. 弹出全屏半透明遮罩与回忆文案。
    4. 温度条 +5。
    5. 玩家再次点击，遮罩消失，恢复行走。

### MemoryPopup 样式 (ASCII)
```text
+--------------------------------+
|                                |
|         (半透明黑色遮罩)       |
|                                |
|  "那天雨很大，我们在这里       |
|     躲了半个小时。"            |
|                                |
|         [ 点击继续 ]           |
+--------------------------------+
```

## 六、 场景结束过渡流程

1. 玩家 X 坐标越过 800px 阈值。
2. 禁用玩家输入，Player 和 NPC 保持现有间距，均以 `NPC_WALK` 速度自动向右走出画面右侧。
3. 摄像机淡出至全黑 `this.cameras.main.fadeOut(500, 0, 0, 0)`。
4. 传递当前数据至下一幕：
   ```typescript
   this.scene.start('CityNightScene', {
       currentTemp: temperatureBar.getValue(),
       distanceOffset: player.x - npc.x
   });
   ```

## 七、 验收标准
- [ ] 角色走动动画正常，摄像机平滑跟随玩家且不越界。
- [ ] 玩家超出 60px 时 NPC 准确停步低头，落后时暗角效果变浓。
- [ ] 走到 350px 和 600px 处，同步状态下能成功触发回忆弹出层。
- [ ] 随着 X 轴推进，画面整体色调能平滑地从暖橙过渡到暗紫。
- [ ] 到达终点后，过渡动画流畅触发并成功携带数据切换至 CityNightScene。
