## 二、 结局 A (不原谅) 演出时间轴

要求严格的 12 秒心碎演出，通过 `Phaser.Time.DelayedCall` 进行编排。

| 时间范围 | 动作描述 | 参数/缓动细节 |
| :--- | :--- | :--- |
| **0 - 0.5s** | 场景淡入 | Alpha: 0 -> 1，信件文字、按钮处于显示状态 |
| **0.5 - 1.0s**| 文字与按钮淡出 | Alpha: 1 -> 0，(Tween，时长 500ms) |
| **1.0 - 1.5s**| NPC 犹豫 | NPC 头顶浮出 `...` 气泡对象 |
| **1.5 - 2.0s**| NPC 起身 | 播放 `npc_sit_stand.png` 帧 1 -> 帧 0 |
| **2.0 - 2.5s**| 站立停顿 | NPC 保持背影/侧面静止，等待 500ms |
| **2.5 - 6.0s**| NPC 离开 | VelocityX: 10 (极慢步行)，向右走出画面边缘 |
| **6.0 - 7.0s**| NPC 渐隐 | NPC 精灵 Alpha: 1 -> 0，模拟走进地铁阴影 |
| **7.0 - 8.0s**| 独坐长镜头 | 主角保持坐姿不动，头顶浮现孤单的 `...` |
| **8.0 - 9.5s**| 画面淡黑 | Camera FadeOut(1500)，同时 BGM 衰减 Volume 1 -> 0 |
| **9.5 - 10.0s**| 纯黑静默 | 屏幕全黑，无任何声音 |
| **10.0 - 11.0s**| 核心句淡入 | 屏幕中央 Y: 160，添加白色文字「也许下次能说出口」，Alpha 0->1 |
| **11.0s 起** | 永久停留 | 停留在当前场景，无重置按钮，强迫玩家感受余韵 |

### 结局 A 实现伪代码

```typescript
function playEndingA() {
    // 隐藏 UI
    this.tweens.add({ targets: [uiGroup], alpha: 0, duration: 500, delay: 500 });

    // 1.0s NPC 犹豫
    this.time.delayedCall(1000, () => showBubble(npc));

    // 1.5s NPC 起身
    this.time.delayedCall(1500, () => npc.play('stand_up'));

    // 2.5s 开始行走离开
    this.time.delayedCall(2500, () => {
        npc.play('walk');
        npc.setVelocityX(10);
    });

    // 6.0s NPC 渐隐
    this.time.delayedCall(6000, () => {
        this.tweens.add({ targets: npc, alpha: 0, duration: 1000 });
    });

    // 8.0s 场景淡出
    this.time.delayedCall(8000, () => {
        this.cameras.main.fadeOut(1500, 0, 0, 0);
        fadeBGMOut(1500);
    });

    // 10.0s 最终文字
    this.time.delayedCall(10000, () => {
        let endText = this.add.text(96, 160, DIALOGUES.ENDING_A_TEXT, { fontSize: '12px' });
        endText.setOrigin(0.5);
        this.tweens.add({ targets: endText, alpha: 1, duration: 1000 });
    });
}
```

## 三、 结局 B (原谅) 临时占位方案

等待最终美术资产到位前，使用粒子系统实现占位演出：
1. **清理 UI**：信件文字与按钮淡出。
2. **星空粒子**：在屏幕上半部分（Y: 0 到 150 区间）发射 `star_particle`。
   * 密度计算：`particleCount = Math.floor(temperature * 2)`。温度越高星星越密集。
3. **爱心浮现**：在两人头顶正上方居中位置，生成一个 `heart_particle`，伴随 Alpha 0->1 和向上的漂移 Tween。
4. **终幕文字**：屏幕淡黑后显示「夜风很冷，但一切刚刚好。」

## 五、 验收标准
- [ ] 结局 A：12 秒时间轴的每一阶段均严格按照设定时长和顺序执行。
- [ ] 结局 A：NPC 行走离开的速度明显慢于正常步行，制造留恋和拉扯感。
- [ ] 结局 A：最终文字出现后，游戏真正处于停止状态，不会再有循环动画干扰。
- [ ] 结局 B：占位粒子的密度能根据传递过来的最终温度值（Temperature）进行动态调整。
- [ ] 确保结局演出过程中，所有交互已被禁用，防止玩家误触导致逻辑崩溃。
